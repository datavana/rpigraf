# Check whether the provided vector contains valid IDs prefixed with table names. Example: articles-123

Check whether the provided vector contains valid IDs prefixed with table
names. Example: articles-123

## Usage

``` r
epi_is_id(ids, table = NA)
```

## Arguments

- ids:

  The vector that will be proofed

- table:

  Check whether the path contains the table. Leave empty to allow all
  tables.

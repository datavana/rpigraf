# Check whether the provided vector contains a valid IRI path

Check whether the provided vector contains a valid IRI path

## Usage

``` r
epi_is_iripath(iripath, table = NA, type = NA)
```

## Arguments

- iripath:

  The vector that will be proofed

- table:

  Check whether the path contains the table. Leave empty to allow all
  tables.

- type:

  Check whether the path contains the type. Leave empty to allow all
  types.

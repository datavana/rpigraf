# Save database connection settings to environment variables. Environment variables are prefixed with "epi\_" and used in db_connect() to establish the connection.

Save database connection settings to environment variables. Environment
variables are prefixed with "epi\_" and used in db_connect() to
establish the connection.

## Usage

``` r
db_setup(
  host = "localhost",
  port = 3306,
  username = "root",
  password = "root",
  database = ""
)
```

## Arguments

- host:

  host

- port:

  port

- username:

  username

- password:

  password

- database:

  database
